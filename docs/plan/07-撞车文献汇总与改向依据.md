# IRIS 相关工作与改向依据（证据对齐版）

> 本文只规定“可以怎样定位”，不把计划中的实验写成结果。引用的版本、会议录用与定量数字在投稿前仍需作者按原文逐项复核。截至 2026-07-13，仓库中的正式 point-level grounding 为 **0**，因而本文不宣称任何 cross-site、cross-action、coupled/decoupled 或 live 模型效果已经得证。

## 1. 不可再宣称的新颖性

IRIS 不应声称发明了下列概念：

- 将世界模型推理写入 agent 的自回归输出；
- 使用前向状态/结果预测帮助 web agent 选择动作；
- 用外置 verifier/guardrail 阻止高风险操作；
- 把“能否回到先前状态”作为安全信号；
- 为 agent 动作建立 reversible/compensable/irreversible 分类；
- 建立一个通用 web-agent safety benchmark。

当前可能的差异点只能写成一个**待验证命题**：在 web agent 的权限、观测信号σ、搜索器集合和预算 k 明确固定时，通过 execute–then–undo 产生逐转移的构造性恢复证据，再检验策略模型能否利用该证据。这不等于证明“可恢复=安全”。

## 2. 与主要相关工作的正确关系

| 工作 | 对 IRIS 的直接约束 | 可以保留的 IRIS 定位 | 当前证据状态 |
|---|---|---|---|
| VAGEN | `<observation>/<prediction>/<answer>` 类显式世界模型 trace、内在化路径与 RL credit assignment 都有直接先例 | IRIS 可将输出格式视为载体，待测的新对象是预算化恢复证据与 undo | 仓库有 formal `iris.v3`/legacy `iris.v2` 格式；无法证明机制泛化的正式数据/结果 |
| WMA / WAC / SafePred | “前向预测用于安全选择”和“外置预测式 guardrail”已有先例 | 不比谁更早用世界模型做安全；只比较前向证据与恢复证据的边际价值 | 无公平 coupled/decoupled 实验 |
| Qwen-AgentWorld / WebWorld | 大规模 next-state/world-simulation 已是独立主线 | IRIS 不将 next-state 建模或 simulator 规模写成贡献 | 历史 SFT 的 `<prediction>` 不等于真实 post-state supervision |
| WebGuard | 规范风险是人/政策对行为的许可性判断，不能被恢复性取代 | WebGuard 的 normative risk 与 IRIS 的 operational recoverability 应当作为正交维度联合报告 | 当前无意见标签对照数据，不能宣称优于 WebGuard |
| ST-WebAgentBench | 策略/约束安全评测已有专门 benchmark | IRIS 可在未来使用其作外部评测，但不把自己包装成新的通用安全 benchmark | 当前未接入该数据/评测 |
| Leave No Trace / Grinsztajn et al. | 可恢复性作为安全信号早于 IRIS | 可研究 web agent 语义动作空间中的显式 undo 序列、代价和残差 | 旧 class-level smoke 记录不构成 point-level benchmark |
| Revisable by Design | 该工作不只是 taxonomy；已包含分类、算法、StreamBench 与真实 agent 实证 | IRIS 只能主张不同的测量单元和行为证据链，且必须通过正式 point-level 数据实证 | 该 delta 尚未由当前数据建立 |

## 3. 核心构念边界

正式对象应写为：

`recoverability(s, a | agent_action_space, privilege, k, σ, solver_set)`

其证据强度是非对称的：

- `RECOVERED`：由已执行的 undo 序列和最终信号复原给出构造性证据；
- `PARTIALLY_RECOVERED`：已执行恢复，但 `residual_diff` 非空；
- `NOT_RECOVERED_WITHIN_BUDGET`：只表示指定 solver set 在 k 步内未恢复，不是数学意义的不可逆；
- `UNKNOWN`：证据、信号或执行不足。

`effect_status` 与 `recovery_status` 必须正交；`NO_EFFECT` 不是一种可逆性。同样，规范安全依赖用户约束、合法性、授权与任务必要性；一个可撤销的动作仍可能是未授权的，一个在声明预算内未恢复的动作也可能是用户明确要求的。

## 4. coupled / decoupled 的诚实定位

coupling 研究目前只是预注册计划，不是已落地贡献。公平对照至少需要：

1. 相同的 point-level labels 和 evidence packet；
2. 相同 base checkpoint；
3. 相同 candidate set 与可见信息；
4. capacity-matched 与 compute-matched 两组对照；
5. 配对记录 tokens、calls、latency、校准和任务结果。

在上述产物不存在时，论文不得使用“首个受控研究”或“内在化优于外置 verifier”的过去时表述。

## 5. 当前贡献声明的证据完备度

| 候选贡献 | 当前可见工程证据 | 能否写成已证贡献 |
|---|---|---|
| 正交 effect/recovery 本体和 point-level schema | 已有版本化 schema、migration 和 validator | 可写为方法/工程设计，不可写为已建成数据集 |
| execute–then–undo 行为测量 | 有 class-level probe smoke 代码和历史记录 | 只可写 pilot/protocol；当前正式 point=0 |
| 恢复性内在化 | 有 assembler、trainer 和格式校验 | 不可；历史 SFT 未精确回链 point，teacher distilled=0 |
| coupled vs decoupled | 无完整成对实验和结果 | 不可 |
| cross-action/cross-site 泛化 | split 代码可在数据足够时生成对应分割 | 不可；当前正式数据和结果都不存在 |

截至本次整改，candidate/point/prompt/truth 的 manifest、联合 train/dev/test split 和 completion 真值一致性 gate 只增强了可复现性；它们不是相对于 VAGEN、WebGuard 或 Revisable by Design 的算法新颖性证据。候选中的 50 条 `source=expert` 仅是 producer-declared target bid，没有成功 collector provenance；历史 DPO 虽按机制为 synthetic flip，385/385 行均缺 `negative_source`。formal point 仍为 0，因此不能把“工程边界更严格”改写成“方法已被实证验证”。

## 6. 论文可用的最小定位

> Prior work has established internalized world-model traces, forward outcome prediction, predictive guardrails, and reversibility as a safety signal. IRIS therefore does not claim any of these ideas in isolation. We study a narrower measurement question: whether a web-agent transition can be behaviorally recovered by a specified agent, privilege level, solver set, observation signal, and search budget, and whether models can use such point-level evidence without conflating recoverability with normative safety. Positive labels are supported by executed recovery traces; negative labels denote failure to recover within the declared budget rather than universal irreversibility.

这段定位只陈述研究问题。在正式 point-level 数据、公平基线和预注册评测完成前，不能将其升格为实证结论。

## 7. 已核对的版本标识

- Wang et al., “VAGEN: Reinforcing World Model Reasoning for Multi-Turn VLM Agents,” [arXiv:2510.16907](https://arxiv.org/abs/2510.16907). 原文明确包含显式 state/transition reasoning、World Modeling Reward 与 Bi-Level GAE。
- Zhai et al., “Revisable by Design: A Theory of Streaming LLM Agent Execution,” [arXiv:2604.23283](https://arxiv.org/abs/2604.23283). 原文明确包含 reversibility taxonomy、Revision Absorber、StreamBench 与真实 agent 实验。
